# USER_GUIDE.md: Инструкция по использованию MVP «Биопротеин»

## Обзор системы
MVP представляет собой аналитическую цепочку для оптимизации процесса ферментации метанотрофных бактерий. Система восстанавливает текущее состояние процесса (ассимиляция), подбирает оптимальный режим разбавления $D(t)$ (оптимизация) и оценивает экономическую эффективность (финмодель).

## Быстрый старт

### 1. Установка зависимостей
Убедитесь, что установлен Python 3.10+.
```bash
pip install -r requirements.txt
```

### 2. Модульное тестирование
Вы можете запустить тесты каждого блока независимо, чтобы убедиться в корректности расчетов:
- **ОДУ модель:** `python test_ode.py` — проверка стабильности решения системы уравнений.
- **Ассимилятор (MND):** `python test_mnd.py` — проверка восстановления параметров из зашумленных данных.
- **Финмодель:** `python test_finmodel.py` — проверка расчета себестоимости и чувствительности к ценам.
- **Оптимизатор:** `python test_optimizer.py` — проверка поиска режима с максимальной прибылью.

### 3. Запуск полного пайплайна
(После реализации `run_pipeline.py`)
```bash
python run_pipeline.py --batch_id <id>
```

## Описание модулей
- `ode_model.py`: Ядро системы. Решает ОДУ для X, S, Y, A, B.
- `mnd_assimilation.py`: Восстанавливает параметры процесса на основе измерений.
- `regime_optimizer.py`: Ищет лучший профиль $D(t)$ с помощью байесовской оптимизации.
- `financial_model.py`: Считает NPV, маржу и себестоимость белка.

## Ограничения
- Вычисления производятся на CPU (ThinkPad).
- Модель предполагает идеальный перемес в струйном ферментере.
- Оптимизация проводится для одного батча (цикла).
